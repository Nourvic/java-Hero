# JavaHero

Ein Einsteiger-Java-Projekt mit Übungsaufgaben und einer kleinen Abschluss-Spielidee.

## Inhalte
- Variablen, Schleifen, Bedingungen
- Methoden, Arrays, Klassen & Objekte
- Konsolenanwendungen
- OOP

# Technologien
- Java 22
- IntelliJ IDEA

# Starten
Projekt in IntelliJ öffnen und Main.java ausführen.
